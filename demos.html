The screencasts and interactive demos below show different aspects of the Phylotastic project
  • Topology Server

    Want to see a phylogeny right now for your user-supplied list of species (e.g., Homo_sapiens,Pan_troglodytes,Gorilla_gorilla)? Check out the web interface to a pruner based on MapReduce technology

  • Mesquite-o-tastic

    What would it be like to get a species phylogeny on-the-fly, without leaving your favorite software workbench for evolutionary analysis?

  • Reconciliotastic

    What if you didn't need to provide your own species tree in order to identify orthologs and paralogs by gene-species tree reconciliation? This live demo shows tree reconciliation by SDI using a Phylotastic species tree.

  • DateLife

    Wouldn't it be great to scale your tree using dates of evolutionary divergence estimated by experts using fossils? Welcome to the live demo of DateLife!

  • Phylo-Taxographer

    Interested in phylogeography? This interative demo uses a taxon phylogeny pruned from ToLWeb to browse global species distribution data from GBIF and Map of Life.

  • Making Galaxy Phylotastic!

    Whether you are a Galaxy user or just curious about what this popular genomics workflow environment can do, this screencast will show how to use Galaxy to compose and execute a workflow that gets a subtree from the mammal supertree, reconciles with NCBI's taxonomy, and retrieves divergence dates from our DateLife server. Want more? Check out the code and the live Galaxy-Phylotastic server.
